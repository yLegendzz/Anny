## ⚙️ Como Usar

<div align="center">
<img src="https://capsule-render.vercel.app/api?type=rect&color=0:c084fc,100:ff85a1&height=3&section=header" width="100%"/>
</div>

### 🚀 Início Rápido

**1. Adicione a Anny ao seu servidor**

Clique no botão abaixo para adicionar a Anny ao seu servidor Discord com todas as permissões necessárias:

<div align="center">
  <a href="https://discord.com/oauth2/authorize?client_id=1071825122467524688&scope=bot%20applications.commands&permissions=2146958847">
    <img src="https://img.shields.io/badge/➕%20ADICIONAR%20A%20ANNY%20AO%20SERVIDOR-5865F2?style=for-the-badge&logo=discord&logoColor=white&labelColor=404EED" alt="Add to Server" height="50"/>
  </a>
</div>

<br/>

**2. Inicie uma conversa**

Assim que a Anny entrar no servidor, basta mencioná-la:

```
@Anny Olá! O que você consegue fazer?
```

**3. Responda às mensagens dela**

A Anny mantém o contexto da conversa. Continue respondendo às mensagens dela para continuar o fio da conversa.

**4. Configure o idioma (opcional)**

```
/configurar
```

### 📖 Comandos Principais

<details>
<summary><b>📋 Ver lista de comandos mais comuns</b></summary>

<br/>

#### 🤖 Comandos de I.A

| Comando | Descrição |
|---|---|
| `@Anny [mensagem]` | Inicia ou continua uma conversa com a I.A |
| `/chat` | Inicie um chat profissional para suporte e assistência |
| `/configurar usuario/anny` | Configura idioma e preferências da conversa, como personalidade, modelo e perfil da Anny |
| `/chatreset` | Limpa o histórico de conversa atual |

#### 🎨 Comandos de Imagem

| Comando | Descrição |
|---|---|
| `/imagem gerar [descrição]` | Gera uma imagem com I.A a partir de um texto |
| `/imagem upscale [imagem]` | Edita uma imagem para uma versão com maior qualidade |

#### 🛠️ Comandos de Servidor

| Comando | Descrição |
|---|---|
| `/sorteio cria` | Cria um sorteio único ou automático no servidor |
| `/embed criar/clonar/editar` | Cria uma mensagem embed personalizada, pode copiar ou editar outras embeds |
| `/limpar` | Limpe mensagens de um canal com vários filtros |

#### 🌍 Comandos Utilitários

| Comando | Descrição |
|---|---|
| `/traduzir [texto] <idiomas>` | Traduz um texto para outros idiomas |
| `/download tiktok/youtube/instagram/... [url]` | Faz download de conteúdo de uma URL |
| `/analisar texto` | Faz uma análise completa de uma mensagem |
| `/botinfo` | Exibe informações gerais sobre a Anny |

</details>

### 🌍 Configurando o Idioma

Use o comando `/configurar` para definir o idioma da conversa com a Anny. Ela suporta os principais idiomas do mundo:

```

/configurar idioma: English
/configurar idioma: Español
/configurar idioma: Français
```

### 💬 Ativando o Chat Automático

Você pode configurar um canal dedicado onde a Anny responde automaticamente a **todas** as mensagens, criando uma experiência de chatbot totalmente integrada ao seu servidor.
